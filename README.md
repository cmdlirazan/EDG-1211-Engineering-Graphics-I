## EDG 1211 – Engineering Graphics I
This repository contains the full set of labs, projects, and assignments I completed independently for **EDG 1211** at **Kennesaw State University**.

### 📂 Assignments
1. **3D CAD Models**
   - Parametric part models created in **SOLIDWORKS**

2. **CAD-Generated Orthographic Drawings**
   - Multi-view drawings generated from SOLIDWORKS models
   - Proper dimensioning and annotations

3. **Components**
   - Individual mechanical components, including hand-drafted sketches, 3D CAD models, and CAD-generated orthographic drawings

4. **Hand-Drafted Orthographic Drawings**
   - Freehand and instrument-assisted drawings
   - Front, top, and side views following ANSI standards

### 🚀 Skills Gained
- Design communication through sketches and CAD drawings
- Orthographic projection and dimensioning
- Parametric 3D modeling in **SOLIDWORKS**
- Technical drawing standards (ANSI)
