# EDG 1211 – Engineering Graphics I

Selected work completed independently for EDG 1211 at Kennesaw State University.

## Features
- Parametric 3D CAD modeling in SOLIDWORKS
- CAD-generated orthographic drawings with proper dimensions and annotations
- Hand-drawn and technical sketches
- Design communication following ANSI standards

## Work Samples

### 3D CAD Models
- Parametric parts created in SOLIDWORKS

### CAD-Generated Orthographic Drawings
- Multi-view drawings generated from CAD models
- Correct dimensions and annotations

### Components
- Individual mechanical components
- Includes hand sketches, 3D CAD models, and CAD-generated drawings

### Hand-Drafted Orthographic Drawings
- Freehand and instrument-assisted sketches
- Front, top, and side views following ANSI standards

## Viewing
All work is saved as PNG images and can be viewed directly in a browser or image viewer.
